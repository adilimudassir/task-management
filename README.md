## Description

This is a practice program i wrote while taking [NestJS](https://www.udemy.com/course/nestjs-zero-to-hero/) course on [Udemy](https://udemy.com) to take my Backend development with NodeJS to the next level.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
